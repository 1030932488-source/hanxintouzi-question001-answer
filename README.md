# 瀚鑫投资笔试题答案

本仓库包含笔试题的所有答案（题目1-4）。

## 目录结构

### 题目 1 & 2 (算法)

- `Question1.md`: 题目1的答案（反向对称矩阵算法）。
- `Question2.md`: 题目2的答案（FFT正则匹配）。

### 题目 3 (后端重构)

- `app/`: 包含重构后的 Flask 应用代码。
- `config.py`: 配置文件。
- `run.py`: 启动脚本。
- `requirements.txt`: 项目依赖。

### 题目 4 (数据获取)

- `main.py`: 数据获取的主程序入口。
- `data_fetcher.py`: 数据获取逻辑。
- `data_storage.py`: 数据存储逻辑。
- `data_validator.py`: 数据校验逻辑。

## 如何运行

1. 安装依赖:

   ```bash
   pip install -r requirements.txt
   ```

2. 运行题目3的后端:

   ```bash
   python run.py
   ```

3. 运行题目4的数据脚本:

   ```bash
   python main.py
   ```
